# ToDo List

Simple static To-Do List web page built with HTML and CSS.

Features
- Static HTML UI for listing tasks with checkboxes.
- CSS styling and images included in the repo root.

Run locally
- Open `index.html` in your browser.
- Or run a simple static server from the project folder:

	```bash
	python -m http.server 8000
	# or
	npx serve .
	```

Deployed
- Live site: https://todo-list-delta-steel.vercel.app

Notes
- Keep `index.html`, `style.css`, and image files at the repository root so Vercel serves them as static assets.

Enjoy! 🎉
